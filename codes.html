Code代码（Code 1-2-3-4-5）

i. Code 1

表示以正常巡逻或无紧急状态方式响应，不使用警灯或警笛。

适用于例行巡逻、非紧急任务。

示例：例行巡逻、普通调度任务。

ii. Code 2

表示以仅开启警灯的方式响应任务，不使用警笛。

可在穿越路口时适度使用气喇叭（Airhorn）确保安全。

通常用于 10-32（请求支援）或 10-11（交通拦截） 等情况，但不需要立即支援。

示例：某警员请求支援，但未遭遇威胁或攻击。

iii. Code 3

表示以开启警灯和警笛的方式响应，是最高紧急等级响应。

用于如下情况：

  • 10-99/Code 99（警员遇险）
  • 10-32 code 3（请求支援）
  • Code 5 停车战术行动
  • 涉及人身危险或生命威胁事件

示例：正在追捕持枪嫌疑人或收到生命危险报告。

iv. Code 4

用于宣告现场"情况已受控"或"无进一步威胁"。

适用于你已完成拘捕、制服嫌疑人、移除武器等，并准备返回正常执勤（10-8）或巡逻状态（Code 1）。

示例："嫌疑人已拘捕，武器收缴，现场 Code 4，单位返回 10-8。"

v. Code 5

这是战术操作用语，详情请参阅 无线电使用：Code 5 战术操作。

vi. Code滥用

"代码滥用"指在非适当场合或无正当理由下使用 Code 2–5 响应方式。

常见错误例如：

  • 为跳过红灯而开启 Code 3（灯光+警笛）；
  • 没有接警指令却私自使用 Code 2/3；
  • 擅自进入他人辖区内使用高等级响应。

提示：你可以在 Code 1 状态下短暂使用 Code 3通过复杂路口（如必要），但必须合理合法。
