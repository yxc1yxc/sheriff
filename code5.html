Code 5 战术操作

i. Code 5 定义：

Code 5 是一种用于逮捕高风险嫌疑人的战术拦截操作，通常适用于：

  • 嫌疑人拥有有效逮捕令；
  • 嫌疑人携带武器或具有潜在严重暴力威胁。

Code 5 的目的是将嫌疑人强制从车辆中带出、跪下、并安全上铐拘捕。

战术部署包括：

  • 至少两辆执法车辆车头对准嫌疑车辆；
  • 所有车辆前门敞开作为掩体；
  • 多名警员同时持有致命武器与非致命武器（如电击枪）；
  • 现场指挥官（通常是最先到场或最早呼叫 Code 5 的单位）负责发布所有指令；

指挥员应持非致命武器（电击枪）并发出以下标准指令：

  • "关闭引擎，双手保持在我们能看见的位置。"
  • "慢慢下车，面朝前方，背对我们。"
  • "听我声音的方向慢慢向后走，走到我说'停'为止。"
  • "停！跪下，保持不动！"

这些是 Code 5 拦截中必须由指挥员对嫌疑人发出的基本指令，必须明确、简洁、无误。

ii. 适用时机：

Code 5 拦截通常应用于以下情境：

高风险交通拦截（10-11）：

初始单位发起呼叫，要求多个增援单位以 Code 3（灯光+警笛） 方式响应并协同部署。

追捕终结阶段（10-80）：

当至少三辆执法车完成对目标车辆的包围拦截，嫌疑人车辆被迫停车，随后立即展开 Code 5 操作。

iii. 交叉火力与团队配合：

在 Code 5 操作中，最关键的安全要素之一是避免交叉火力以及保持高效团队协作：

  • 执勤过程中，即便你与其他队员私下有竞争关系，在战术行动中必须无条件优先服从团队配合与安全原则；
  • 你必须随时注意你的射界范围，避免队员进入你武器的 45 度视野角内；
  • 绝不可将武器对准同僚（无论是警员、巡警还是警长），包括致命武器与非致命武器（如电击枪）；
  • 始终保持对队友位置与计划的认知，提前沟通行动方向——"思维同步"是行动成功的关键。
